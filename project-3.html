Project Overview

Southtech Electric wanted a site that showcased their services while staying easy to update. I used Astro with Codestitch's component library, converting components into Astro for a modular, maintainable codebase.

The result is a responsive site that works seamlessly across devices and provides a professional presence for the client. This project demonstrates my ability to deliver real-world websites for small businesses on time and with long-term maintainability in mind.
